
#  Conversor de Temperaturas (Python)

Este es un programa sencillo escrito en Python que permite convertir temperaturas entre dos escalas:

- Celsius → Fahrenheit  
- Fahrenheit → Celsius  

Es uno de mis primeros proyectos para comenzar mi portafolio de desarrollo en Python.

---

## Funcionalidades

- Menú interactivo por consola  
- Conversión precisa utilizando las fórmulas oficiales  
- Validación básica de opción ingresada  
- Interfaz simple y fácil de usar  

---

## ¿Cómo ejecutar el programa?

Clonar o descargar este repositorio y ejecutar en terminal:

```bash
python conversor_temperaturas.py
